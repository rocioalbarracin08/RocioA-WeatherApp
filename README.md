App mobile hecha con Expo y React Native que muestra el clima del día actual, anterior y día siguiente, según tu ubicación.

## Descargar

- APK (Android):
https://github.com/rocioalbarracin08/RocioA-WeatherApp/releases

## Instalación (Android)

1. Descargar la APK desde Releases
2. Abrir el archivo .apk
3. Aceptar “orígenes desconocidos” si lo pide
4. Instalar

## Tener en cuenta...

- Es necesario estar conectado internet
- La APK incluye API Key para la conexión con la api de [weatherApi](https://www.weatherapi.com/)
